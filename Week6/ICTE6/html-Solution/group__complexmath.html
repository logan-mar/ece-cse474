perform_operation

This function performs an operation specified by the operation_t enum on two complex numbers. If the operation is not recognized, it returns a complex number with NaN for both real and imaginary parts.

Here is a usage example:

complex_t a = {1, 2};
complex_t b = {3, 4};
complex_t result = perform_complex_operation(a, b, ADD);
